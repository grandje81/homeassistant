# homeassistant
Documentation for setting up homeassistant on SBC (rPi 5) and KVM

## Hardware used in this setup

- An SBC in form of a Raspberry Pi 5 with 16 GB RAM
- Pimoroni NVMe Base
- NVMe SSD with 500GB (ADATA, that followed the Pimoroni purchase)
- WaveShare Poe+ (G) board
- Raspberry Pi 5 SKU Cooler
- SMLight SLZB-07p7 with Zigbee Firmware
- SMLight SLZB-07p7 with Thread Firmware

Its assumed that an existing PoE+ Network Unit is available to power the rPi 

## Raspberry Pi 4

The setup have also been tested with a rPi 4 with the following hardware
- 8GB of RAM
- Argon One Case
- SSD connected through SATA3-2-USB connector.
- Extension cables for Zigbee/Matter Controllers
